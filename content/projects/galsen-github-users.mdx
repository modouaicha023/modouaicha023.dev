**Galsen GitHub Users** is an open source web application that showcases GitHub profiles of Senegalese developers — highlighting their favorite programming languages, contributions, and activity levels.

The project aims to promote the Senegalese tech community and inspire collaboration, learning, and visibility for developers across the region and the diaspora.
